# crawldeck V1 Tasks

## Done in initial OSS Factory build

- Scaffold TypeScript OSS CLI package from StackForge.
- Keep all state local in `.crawldeck/queue.json`.
- Support crawl profiles with fixture adapter configuration.
- Support job enqueue, list, pause, resume, start, and complete commands.
- Generate health and markdown report output.
- Provide adapter registry seams for future real crawlers.
- Ship deterministic fixtures and fixture-backed tests.
- Add CLI smoke script and local validation path.
- Document safety, privacy, attribution, and examples.

## Later
